# Conway's Game of Life

An interactive web-based simulation of Conway's Game of Life, a cellular automaton devised by John Conway in 1970. Features include real-time controls, adjustable speed and grid size, and live statistics tracking.

## Getting Started

Simply open [`index.html`](index.html) in your web browser - no installation required. Click cells to toggle them, then use the Start/Pause/Step buttons to control the simulation.

## Rules

- Live cells with 2-3 neighbors survive
- Dead cells with exactly 3 neighbors become alive
- All other cells die or remain dead
